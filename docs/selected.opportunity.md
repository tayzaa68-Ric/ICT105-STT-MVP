# Selected Semester Project Opportunity
## Selected Project Title

Student Project Management System
## Target User

University students working on individual assignments and group projects.

## Problem Statement

University students often struggle to manage project tasks, assignment deadlines, project files, and team communication. Important information is usually spread across chat groups, documents, and personal notes, making it difficult to stay organized and meet deadlines.

## Current Alternative

Students currently use:

- LINE Groups
- Google Drive
- Personal Notes
- Calendar Applications
- These alternatives are useful but are not integrated into a single platform.

## Proposed IT Solution

A web-based Student Project Management System that allows students to:

- Create projects
- Assign tasks
- Track deadlines
- Monitor project progress
- Store project resources
- Improve team communication

## Why This Opportunity Was Selected

The project received the highest NUF score because it is useful, feasible, and solves a real problem experienced by university students. The system can be developed using available skills and tools during the semester.

### NUF Scoring

| Criteria | Score |
|----------|-------|
| New | 5 |
| Useful | 5 |
| Feasible | 5 |
| Total | 15 |

## Why This Is Feasible Without IoT or Advanced Cybersecurity

The MVP only requires HTML, CSS, JavaScript, GitHub, and simple web technologies. No IoT hardware, sensors, or advanced cybersecurity infrastructure are required.

## Possible MVP Type

HTML/CSS/JavaScript Web Prototype

## Expected Benefits

- Better project organization
- Improved teamwork
- Reduced missed deadlines
- Easier file management
- Increased student productivity

## Final Decision

The team officially selected the Student Project Management System as the semester MVP project for ICT105.
